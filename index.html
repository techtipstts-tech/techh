你會學到什麼

從觀念到實作，一次補齊

A. 觀念建立

  • 流程設計：如何判斷可自動化、如何定義輸入/輸出。
  • 資料結構：JSON、欄位對應、常見陷阱。
  • 部署策略：雲端（Zeabur）與本機的取捨。

B. n8n 核心

  • 必備節點：Webhook / Set / If / HTTP Request。
  • 與 ChatGPT 串接：摘要、分類、結構化輸出。
  • 排錯心法：分支、重試、錯誤通知。

C. 進階與維運

  • 版本控制、備份、觀察指標。
  • 跨平台整合：ChatGPT / Meta / line API 串接概念。
  • 多模型應用與 RAG 入門。

D. 誰適合

  • 零基礎新手、個人工作者、中小企業營運/行銷/客服。
  • 希望把重複流程自動化、建立可持續的工作流的人。
