從 0 到上線

LINE × n8n × Google：快速串接教學

A. 建立 LINE Messaging API

  1. 到 LINE Developers 建立 Channel（Messaging API）。
  2. 開啟 Webhook，將 n8n Webhook URL 貼上並驗證。
  3. 複製 Channel access token，在 n8n 設定 HTTP Bearer 憑證（用於抓圖與回覆）。
  • 抓圖片：GET https://api-data.line.me/v2/bot/message/{messageId}/content
  • 回覆訊息：POST https://api.line.me/v2/bot/message/reply（Body 需帶 replyToken）

B. n8n 流程骨架（與範本一致）

  • Webhook：接 LINE 事件，取得 replyToken 與 message.id
  • HTTP Request（抓圖）：用 Bearer Token 取回圖片（binary）
  • Google Drive：以 message.id.jpg 存入指定資料夾
  • AI 節點：提示詞限制輸出固定 JSON（號碼／日期／店家／金額）
  • Set / If：清洗欄位與缺漏檢查（例如金額只留數字）
  • Google Sheets：appendOrUpdate 寫入欄位
  • HTTP Request（回覆）：把結果文字回傳 LINE（帶 replyToken）

C. 憑證與欄位設定

  • LINE：HTTP Bearer 憑證（Channel access token）。
  • Google Drive／Sheets：用 OAuth2 綁定；指定資料夾與試算表。
  • AI 模型：OpenAI Key；若無法使用，改用 Gemini／Claude／Perplexity（替換節點即可）。

D. 常見錯誤 & 快速排除

  • 401／403：Bearer Token 或 Google 憑證未綁對 → 重新驗證。
  • 圖片未存：Drive 節點需選正確 binary 欄位；檔名可用 {{ $json.body.events[0].message.id }}.jpg。
  • AI 非 JSON：提示只允許 JSON；後面加 Function 做 try/catch。
  • 金額亂碼：用 Set + 正則只留數字與小數點。
